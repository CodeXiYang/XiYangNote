# Git常用命令复习

```shell
# =====================================单机常用操作
# 初始化文件夹为git仓库
git init
# 将工作区中的代码提交到暂缓区
git add .
# 将暂缓区中的代码提交到分支上
git commit -m"描述"
# 查看文件的修改状态
git status



# =====================================联网常用操作
# 将本地仓库中的代码推送到远程仓库上
git push
# 将远程仓库上的代码更新到本地
git pull
```
